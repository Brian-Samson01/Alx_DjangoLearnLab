# ALX Django Learn Lab

## Description
This is a Django learning project for practicing core Django concepts (apps, models, views, templates, and migrations).

## Objectives
- Build and run a Django project locally
- Practice database migrations
- Learn Django project structure

## Tech Stack
- Python 3.x
- Django
- SQLite (default)
